# Agent 002 - Java Expert Agent

## Overview

Java Expert Agent is a specialized AI agent that behaves like a Senior Java Architect with extensive experience in enterprise software development.

The agent is designed to provide practical guidance, architectural recommendations, and mentoring for Java developers.

## Agent Persona

Role:
Senior Java Architect

Expertise:

* Core Java
* Spring Boot
* Microservices
* Design Patterns
* Enterprise Architecture
* Performance Tuning
* System Design

## Architecture

User → Java Expert Agent → Gemini API → Expert Response

## Features

* Java-focused responses
* Enterprise-level explanations
* Architectural guidance
* Spring Boot best practices
* Design pattern recommendations
* Interactive terminal chat

## Technologies Used

* Python 3.13
* Gemini API
* Google GenAI SDK
* VS Code

## Agent Design

This project introduces the concept of:

* Agent Personas
* Context Engineering
* Prompt Engineering
* Role-Based AI Agents

The agent works by combining:

1. System Prompt
2. User Question
3. Gemini Model Response

to create specialized behavior without requiring a custom model.

## Example Questions

* Explain Dependency Injection.
* How should I design a Spring Boot microservice?
* When should I use Factory Pattern?
* How do I improve JVM performance?
