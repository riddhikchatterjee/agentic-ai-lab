Purpose:
Build a menu-driven Tool Agent that can execute different actions based on user selection.

The goal is to introduce the concept of tools and actions, which are fundamental building blocks of Agentic AI systems. This agent demonstrates how user requests can be routed to different functions (tools) and how tool outputs can be returned to the user.
