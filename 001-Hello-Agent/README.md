# Agent 001 - Hello Agent

## Overview

Hello Agent is my first AI agent project built as part of my journey from Java Engineer to AI Engineer.

The purpose of this project is to establish the foundation for future AI agent development by learning:

* Python Virtual Environments
* Gemini API Integration
* SDK Usage
* Prompting
* AI Application Structure

## Architecture

User → Python Application → Gemini API → Response

## Features

* Accepts user questions from the terminal
* Sends prompts to Gemini
* Displays AI-generated responses
* Supports continuous interaction through a command-line interface

## Technologies Used

* Python 3.13
* Gemini API
* Google GenAI SDK
* VS Code

## Key Learnings

* Setting up Python virtual environments
* Working with API keys
* Making LLM API calls
* Understanding SDKs and REST APIs
* Building a simple AI application

## Future Enhancements
